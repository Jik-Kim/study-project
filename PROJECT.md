# PROJECT

## 한 줄 정의

손 제스처로 추적 상태를 변경하고, 색상 객체 위치를 기반으로 ROS2 시뮬레이션
로봇을 제어한다.

## 기술 방향

- Python 3.12와 ROS2 Jazzy
- `rclpy` 기반 애플리케이션 노드
- `rosidl` 기반 공통 인터페이스
- OpenCV HSV 기반 색상 객체 추적
- MediaPipe 기반 손 제스처 인식
- turtlesim 우선 적용

## 역할

| 담당자 | 영역 |
|---|---|
| 김병직 | OpenCV 객체 추적 |
| 조정묵 | MediaPipe 손 제스처 |
| 황인재 | 제어 알고리즘 |
| 이시율 | ROS2 통신 및 인터페이스 |
| 김도윤 | 통합, 시각화 및 검증 |

세부 결정과 미결정 사항은 `docs/SOT.md`를 따른다.
