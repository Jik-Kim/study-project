# 코드 작성 규칙 (Coding Convention)

-   변수명은 영어 사용
-   함수명은 snake_case
-   클래스명은 PascalCase
-   상수는 UPPER_CASE
-   주석은 한국어
-   함수 하나는 하나의 기능만 수행
-   함수 길이는 가능하면 50줄 이하
-   들여쓰기는 스페이스 4칸

  항목       규칙
  ---------- --------------
  변수       snake_case
  함수       snake_case
  클래스     PascalCase
  상수       UPPER_CASE
  주석       한국어
  들여쓰기   스페이스 4칸
